Contributing to CodeGrade¶

By contributing to this product you agree that all code you write will be released under the AGPL license, version 3. Furthermore, all communication should be done in English.

Please consult our Github Page for more information about contributing to CodeGrade (see CONTRIBUTING.md).

Bugs and Suggestions¶

We strive to make CodeGrade the best solution for programming education. If you encounter an bugs in the CodeGrade systems or if you have suggestions to make CodeGrade even better, please feel free to:

  • Submit an Issue on our Github Issues page.

  • E-mail us at info@codegra.de.

We highly appreciate taking your time to inform us about bugs or suggestions.
